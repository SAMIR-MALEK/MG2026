## 🚀 التشغيل خطوة بخطوة

### ✅ المتطلبات
- Node.js v18 أو أحدث: https://nodejs.org
- Git: https://git-scm.com

---

### 1️⃣ إعداد قاعدة البيانات (Supabase — مجاني)

1. اذهب إلى https://supabase.com → **Sign Up**
2. اضغط **New Project** واختر اسماً وكلمة مرور
3. بعد الإنشاء: **Settings → Database → Connection string → URI**
4. انسخ الرابط (يبدو هكذا):
   ```
   postgresql://postgres:[PASSWORD]@db.xxxxx.supabase.co:5432/postgres
   ```

---

### 2️⃣ تشغيل Backend

```bash
# ادخل مجلد الخادم
cd backend

# ثبّت المكتبات
npm install

# أنشئ ملف الإعدادات
copy .env.example .env
```

افتح ملف `backend/.env` وعدّله:
```env
DATABASE_URL="الرابط الذي نسخته من Supabase"
JWT_SECRET="any-long-secret-string-2025"
PORT=5000
FRONTEND_URL="http://localhost:5173"
```

```bash
# أنشئ جداول قاعدة البيانات
npm run db:generate
npm run db:push

# أضف البيانات الأولية (مستخدمين + وسائل + مكاتب)
npm run db:seed

# شغّل الخادم
npm run dev
```

**✅ ستظهر: `الخادم يعمل على المنفذ 5000`**

---

### 3️⃣ تشغيل Frontend (نافذة جديدة)

```bash
cd frontend
npm install
copy .env.example .env
npm run dev
```

افتح المتصفح على: **http://localhost:5173**

## 🌐 النشر على الإنترنت

### Backend → Railway
1. https://railway.app → New Project → GitHub
2. اختر مجلد `backend` كـ Root Directory
3. أضف متغيرات البيئة (DATABASE_URL, JWT_SECRET...)
4. انسخ رابط Railway

### Frontend → Vercel
1. https://vercel.com → New Project → GitHub
2. اختر مجلد `frontend` كـ Root Directory
3. أضف: `VITE_API_URL=https://[railway-url]/api`
4. Deploy

## 📋 دليل استخدام الجرد اليدوي

1. من القائمة → **الجرد اليدوي**
2. اضغط **فتح ورقة جرد** واختر الموقع والتاريخ
3. أدخل **الكمية الفعلية** في كل خلية
4. الفروق تظهر تلقائياً:
   - ✅ أخضر = متطابق
   - 🔴 أحمر = ناقص
   - 🔵 أزرق = زيادة
5. اضغط **إغلاق الجرد** → يُنشأ **محضر رسمي** للطباعة

---

## ▦ دليل طباعة الباركود بـ XPrinter

1. من القائمة → **الباركود والطباعة**
2. اختر المقاس: **58mm** أو **80mm**
3. اختر نوع الباركود: **CODE 128** (موصى به)
4. حدد الوسائل المراد طباعتها
5. اضغط **طباعة** → اختر طابعة XPrinter

> 💡 الوسائل فوق 500 دج فقط تحمل باركود تلقائياً

---

## 📨 دليل نظام الطلبات

**مسؤول المكتب:**
1. من القائمة → **طلباتي** → **طلب جديد**
2. أدخل الوسائل المطلوبة والسبب
3. اضغط **إرسال** — سيصل الطلب لمسؤول الوسائل

**مسؤول الوسائل:**
1. من القائمة → **طلبات المصالح**
2. راجع الطلب واضغط **تحديد موعد الاستلام**
3. حدد التاريخ والساعة والمكان
4. اضغط **تأكيد وإرسال الإشعار**
5. يُشعَر مسؤول المكتب بموعد الاستلام تلقائياً

## 🆘 حل مشاكل شائعة

| المشكلة | الحل |
|---------|------|
| `Cannot connect to database` | تحقق من DATABASE_URL في .env |
| `401 Unauthorized` | شغّل `npm run db:seed` أولاً |
| الواجهة لا تتصل بالخادم | تحقق من VITE_API_URL في frontend/.env |
| الخادم لا يبدأ | تحقق من تثبيت Node.js v18+ |
